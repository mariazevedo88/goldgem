Simple framework that implements some basic design patterns for Java projects using MVC.

## Dependencies

- Java 8
- jUnit 5 (org.junit.jupiter.api, org.junit.jupiter.engine and org.junit.jupiter.params - 5.4.0)
- Hibernate (5.4.4 Final)
- MySQL (driver 8.0.17)
- Log4j (1.2.17)
- Joda Time (2.10.3)
- Coveralls (4.3.0)
- Jacoco (0.7.9)
